# 配对流程图解 / Pairing Flow Diagram

## 整体流程 / Overall Flow

```
┌─────────────────┐         ┌─────────────────┐         ┌─────────────────┐
│   飞虾 App       │         │   配对插件       │         │   OpenClaw      │
│  (手机端)        │         │  (电脑端)        │         │  (网关)         │
└────────┬────────┘         └────────┬────────┘         └────────┬────────┘
         │                           │                           │
         │  1. 生成 Claim            │                           │
         │  (deviceId + publicKey)   │                           │
         │───────────────────────────>│                           │
         │                           │                           │
         │  2. 显示配对指令           │                           │
         │  (复制/分享)              │                           │
         │<───────────────────────────│                           │
         │                           │                           │
         │                           │  3. 粘贴到电脑执行          │
         │                           │  openclaw pair-android     │
         │                           │───────────────────────────>│
         │                           │                           │
         │                           │  4. 生成 Setup Code        │
         │                           │  (bootstrap tokens)        │
         │                           │<───────────────────────────│
         │                           │                           │
         │  5. 轮询获取 Setup Code   │                           │
         │  GET /device-pair/claims  │                           │
         │───────────────────────────>│                           │
         │                           │                           │
         │  6. 返回 Setup Code       │                           │
         │<───────────────────────────│                           │
         │                           │                           │
         │  7. 自动完成配对           │                           │
         │  (WebSocket 连接)         │                           │
         │───────────────────────────────────────────────────────>│
         │                           │                           │
         │  ✅ 配对成功！             │                           │
         │<───────────────────────────────────────────────────────│
```
